### This is a flask app with several payment methods implemented.

<img src="static/IMG/webpage.png" alt="webpage" width="300"/>


to run the app use:

`git clone https://github.com/kpashko/flaskshop`

`cd flaskshop`

`pip install pipenv`

`pipenv install`

`pipenv shell`

`flask run`
